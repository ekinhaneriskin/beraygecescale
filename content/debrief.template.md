# Debrief / Post-Participation Text Template

Use this file for the post-submission message. Keep it general and free of sensitive information.

## Placeholder fields

- `[THANK_YOU_MESSAGE]`: Short thank-you message.
- `[STUDY_SUMMARY]`: General summary of the study.
- `[DATA_USE_STATEMENT]`: Explain that data will be analyzed in aggregate.
- `[RESULTS_ACCESS_INFORMATION]`: Explain how participants can request summary results, if applicable.
- `[CONTACT_INFORMATION]`: Public/corresponding-author contact only if intended for publication.

## Example text

`[THANK_YOU_MESSAGE]`

This study examines `[STUDY_SUMMARY]`.

Your responses will be analyzed in aggregate and reported without identifying individual participants: `[DATA_USE_STATEMENT]`.

For summary results or study-related questions, contact: `[CONTACT_INFORMATION]`.
