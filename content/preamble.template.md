# Preamble / Participant Information Template

Use this file for the public pre-survey information text. Replace placeholders with your own study information.

## Placeholder fields

- `[SURVEY_TITLE]`: Public title of the survey.
- `[RESEARCH_PURPOSE]`: Brief, non-sensitive study purpose.
- `[APPROXIMATE_DURATION]`: Estimated completion time.
- `[TASK_DESCRIPTION]`: Brief description of what participants will do.
- `[VOLUNTARY_PARTICIPATION_STATEMENT]`: Statement that participation is voluntary.
- `[RISK_BENEFIT_STATEMENT]`: Brief risk/benefit statement.
- `[PRIVACY_ANONYMITY_STATEMENT]`: Explain what data are collected and whether responses are anonymous/anonymized.
- `[ETHICS_APPROVAL_STATEMENT]`: Include only if you want the public template to show ethics information.
- `[CONTACT_INFORMATION]`: Use public/corresponding-author contact only if intended for publication.

## Example text

`[SURVEY_TITLE]`

This survey examines `[RESEARCH_PURPOSE]`.

Participation takes approximately `[APPROXIMATE_DURATION]`. You will complete `[TASK_DESCRIPTION]`.

Participation is voluntary. You may stop at any time without penalty.

Responses will be handled according to `[PRIVACY_ANONYMITY_STATEMENT]`.

Ethics approval: `[ETHICS_APPROVAL_STATEMENT]`.

Contact: `[CONTACT_INFORMATION]`.
